## Usage

### Installation

```bash
npm install
```

### Development

Run and visit http://localhost:5173

```bash
npm run start:dev
```

### Build

To build the App, run

```bash
npm run build
```

### Preview

To preview the App, run

```bash
npm run preview
```

### Test

```bash
npm test
```
